# 系统架构设计

## 四层闭环架构

```
┌───────────────────────────────────────────────────┐
│ Layer 4 — 远程监控联动层                           │
│ ├─ 钉钉推送 (alert_dispatcher)                     │
│ ├─ Web 面板 (Flask + Rosbridge)                    │
│ └─ 数据存储 (SQLite + MinIO)                       │
├───────────────────────────────────────────────────┤
│ Layer 3 — 功能执行层                               │
│ ├─ 自主巡逻 (patrol_manager 状态机)                 │
│ ├─ 异常检测 (patrol_yolo YOLOv5n)                  │
│ └─ 报警联动 (patrol_alert 多通道)                   │
├───────────────────────────────────────────────────┤
│ Layer 2 — 数据处理层                               │
│ ├─ 传感融合 (robot_localization EKF)               │
│ ├─ 定位建图 (slam_toolbox + AMCL)                  │
│ └─ 路径规划 (Nav2: A* + DWB)                       │
├───────────────────────────────────────────────────┤
│ Layer 1 — 硬件感知层                               │
│ ├─ RPi5 8GB (主控)                                 │
│ ├─ YDLIDAR X4 / Astra Pro / IMU (传感器)           │
│ └─ ERF01 STM32 (电机驱动)                          │
└───────────────────────────────────────────────────┘
```

## ROS2 节点拓扑

```
  [camera_node]      [lidar_node]      [imu_node]
       │                  │                │
       ▼                  ▼                ▼
  /camera/rgb       /scan            /imu/data
       │                  │                │
       │     ┌────────────┴─────────────┐  │
       │     │   robot_localization     │  │
       │     │   (EKF 融合)             │──┘
       │     └────────────┬─────────────┘
       │                  │
       │     ┌────────────▼─────────────┐
       │     │   slam_toolbox / AMCL    │
       │     │   (建图 + 定位)           │
       │     └────────────┬─────────────┘
       │                  │
       │     ┌────────────▼─────────────┐
       │     │   Navigation2            │
       │     │   (路径规划 + 避障)        │
       │     └────────────┬─────────────┘
       │                  │
       ▼                  ▼
  [patrol_yolo]    [patrol_manager]
  /detected_       /patrol_state
   objects              │
       │                 │
       └────┬────────────┘
            ▼
     [patrol_alert]
     → 钉钉 Webhook
     → GPIO 蜂鸣器
```

## 巡逻状态机

```
NORMAL → 到达预置点 → 360° 扫描
  ↑                       │
  │                ┌──── 检测到异常？── 否 ──┐
  │                ▼                        │
  │           TRACKING (跟踪 10s)            │
  │                │                        │
  │          ┌── 持续异常？── 否 ────────────┘
  │          ▼
  │         ALERT
  │     ├─ 声光报警 (GPIO)
  │     ├─ 钉钉推送 (Webhook)
  │     ├─ 抓拍存证 (JPG → SSD)
  │     └─ 继续巡逻 ──────────┘
```
